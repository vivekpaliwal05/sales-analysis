# 🛒 Grocery Sales Analysis Dashboard

This project focuses on analyzing grocery sales data using **Excel and Power BI** to generate meaningful business insights and track sales performance.

## 📊 Project Overview

The Grocery Sales Analysis Dashboard provides an interactive view of sales trends, product performance, customer purchasing patterns, and revenue insights.

The project includes:
- Raw sales data in Excel format
- Data cleaning and transformation
- Interactive Power BI dashboard
- Business performance analysis

## 🛠️ Tools & Technologies Used

- Microsoft Excel
- Power BI
- Power Query
- DAX
- Data Visualization
- Data Cleaning & Transformation

## 📌 Dashboard Features

- Total Sales and Revenue Tracking
- Product Category Analysis
- Top-Selling Grocery Products
- Sales Trend Analysis
- Customer Purchase Insights
- Region-wise Sales Performance
- Interactive Filters and KPIs
